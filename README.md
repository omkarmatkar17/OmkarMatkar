<b>🔍 LLM Usage Auditor (Master's Capstone)</b><br>
Companies are struggling to see who is using AI tools on their networks. As part of a team capstone, I built the policy enforcement side of a system that detects and controls LLM usage across Windows endpoints, using PowerShell and Windows Firewall to turn detection into action.<br>
➡️ <a href="https://github.com/Rogue-LLM-MQ">github.com/Rogue-LLM-MQ</a>

<br><br>

<b>🌐 LLM Blocker — Chrome Extension</b><br>
A browser extension that spots when someone tries to use an LLM in the browser and blocks it on the spot. Built as the endpoint layer to go with the network controls above.<br>
➡️ <a href="https://github.com/Rogue-LLM-MQ/LLM-Blocker-Chrome">LLM-Blocker-Chrome</a>
